About
It can be hard choosing your home for the next year or so based only on a few pictures. Similar to openigloo, DartHouse is an effort to give renters in the Dartmouth College area more information before they make their choice. All reviews are anonymous, and this website is completely non-profit.

While this website was made to be as clear, simple, and friendly-looking as possible, I am not a professional website designer, so there might be some things that can be improved. Please email me, Nsomma Alilonu, at nsomma_alilonu@yahoo.com to tell me about your concerns, and I'll try to get to them when I can! I'm an M1 starting in August 2022, so it may be a while before I can make any changes to the website; however, I will do my best! If you are interested in making contributions to the website yourself, the source code can be found here. Happy house hunting!
